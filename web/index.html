ArenAI Grid – Live

Kick off a match from the browser and watch it play out.
Quick primer (what you’re seeing)
Goal: Units harvest resources (1/2/3) and return to base for points; attacking can kill units (+10) and eventually destroy the enemy base (+30).
Turn flow: Both AIs take turns; each unit acts once (move, harvest, attack, or idle).
Board legend: Blue/Red bases (B/R), units (b/r), obstacles (#), resources (1/2/3).
HUD: Scores, base HP, units alive, recent events. Combo bonus when a drop has at least one of each resource type.
